# useFetchThread

The `useFetchThread` hook allows components to fetch thread id from ACS resource and store it in ChatThreadProvider. The
hook returns a function of the form `() => Promise<void>)`.

## Importing

```javascript
import { useFetchThread } from '@azure/communication-ui';
```

## Dependencies

- ChatProvider
- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import {
  ChatProvider,
  ChatThreadProvider,
  useFetchThread
} from '@azure/communication-ui';

const Example = () => (
  <ChatProvider config={token: <TOKEN>, displayName: <DISPLAYNAME>, endpointUrl: <URL>, threadId: <THREADID>}>
    <ChatThreadProvider>
      <App />
    </ChatThreadProvider>
  </ChatProvider>
);

const App = () => {
  const fetchThread = useFetchThread();
  fetchThread();
}
```
