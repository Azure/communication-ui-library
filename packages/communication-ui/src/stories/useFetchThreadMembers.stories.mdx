# useFetchThreadMembers

The `useFetchThreadMembers` hook allows components to fetch thread members from ACS resource and store them in
ChatThreadProvider. The hook returns a function of the form `() => Promise<void>)`.

## Importing

```javascript
import { useFetchThreadMembers } from '@azure/communication-ui';
```

## Dependencies

- ChatProvider
- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import {
  ChatProvider,
  ChatThreadProvider,
  useFetchThreadMembers
} from '@azure/communication-ui';

const Example = () => (
  <ChatProvider config={token: <TOKEN>, displayName: <DISPLAYNAME>, endpointUrl: <URL>, threadId: <THREADID>}>
    <ChatThreadProvider>
      <App />
    </ChatThreadProvider>
  </ChatProvider>
);

const App = () => {
  const fetchThreadMembers = useFetchThreadMembers();
  fetchThreadMembers();
}
```
