# useFetchReadReceipts

The `useFetchReadReceipts` hook allows components to fetch read receipts from ACS resource for the current thread
managed by ChatThreadProvider. The hook returns a function of the form `() => Promise<ReadReceipt[]>`.

## Importing

```javascript
import { useFetchReadReceipts } from '@azure/communication-ui';
```

## Dependencies

- ChatProvider
- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import {
  ChatProvider,
  ChatThreadProvider,
  useFetchReadReceipts
} from '@azure/communication-ui';

const Example = () => (
  <ChatProvider config={token: <TOKEN>, displayName: <DISPLAYNAME>, endpointUrl: <URL>, threadId: <THREADID>}>
    <ChatThreadProvider>
      <App />
    </ChatThreadProvider>
  </ChatProvider>
);

const App = () => {
  const fetchReadReceipts = useFetchReadReceipts();
  const readReceipts = fetchReadReceipts();
}
```
