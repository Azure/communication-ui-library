# useFetchMessages

The `useFetchMessages` hook allows components to fetch lists of messages from ACS resource for the current thread
managed by ChatThreadProvider. The hook returns a function of the form
`(options?: ListMessagesOptions) => Promise<ChatMessage[]>`.

## Importing

```javascript
import { useFetchMessages } from '@azure/communication-ui';
```

## Dependencies

- ChatProvider
- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import {
  ChatProvider,
  ChatThreadProvider,
  useFetchMessages
} from '@azure/communication-ui';

const Example = () => (
  <ChatProvider config={token: <TOKEN>, displayName: <DISPLAYNAME>, endpointUrl: <URL>, threadId: <THREADID>}>
    <ChatThreadProvider>
      <App />
    </ChatThreadProvider>
  </ChatProvider>
);

const App = () => {
  const fetchMessages = useFetchMessages();
  const messages = fetchMessages();
}
```
