# useUpdateThreadTopicName

The `useUpdateThreadTopicName` hook returns a callback of the form `(topicName: string) => Promise<boolean>`. The
callback should be stored and then invoked when you want to change the topic name of a chat thread. The callback will
invoke the underlying Chat SDK which means the change will be persisted to the ACS resource.

## Importing

```javascript
import { useUpdateThreadTopicName } from '@azure/communication-ui';
```

## Dependencies

- ChatProvider
- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import {
  ChatProvider,
  ChatThreadProvider,
  useUpdateThreadTopicName
} from '@azure/communication-ui';

const Example = () => (
  <ChatProvider config={token: <TOKEN>, displayName: <DISPLAYNAME>, endpointUrl: <URL>, threadId: <THREADID>}>
    <ChatThreadProvider>
      <App />
    </ChatThreadProvider>
  </ChatProvider>
);

const App = () => {
  const updateThreadTopicName = useUpdateThreadTopicName();
  updateThreadTopicName('new thread topic name');
}
```
