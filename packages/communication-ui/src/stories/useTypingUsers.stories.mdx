# useTypingUsers

The `useTypingUsers` hook allows components to subscribe to typing notifications and provides a list of users that are
currently typing. It internally manages and checks when users stop/start typing and returns an appropriate list by
triggering the current component to re-render. The hook also takes in a list of ChatThreadMembers which is used to match
against typing users. This list should be updated if list of members changes. By default if a typing notification is
received for a user, that user will be returned as typing by useTypingUsers for 8 seconds (not currently change-able).

## Importing

```javascript
import { useTypingUsers } from '@azure/acs-ui-sdk';
```

## Dependencies

- ChatProvider
- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import {
  ChatProvider,
  ChatThreadProvider,
  useTypingUsers
} from '@azure/acs-ui-sdk';

const Example = () => (
  <ChatProvider config={token: <TOKEN>, displayName: <DISPLAYNAME>, endpointUrl: <URL>, threadId: <THREADID>}>
    <ChatThreadProvider>
      <App />
    </ChatThreadProvider>
  </ChatProvider>
);

const App = () => {
  const typingUsers = useTypingUsers([]);
}
```
