# useOutgoingCall

The `useOutgoingCall` hook allows components to leverage the underlying Calling JS SDK to make a 1-to-1 voip call.

## Importing

```javascript
import { useOutgoingCall } from '@azure/acs-ui-sdk';
```

The hook depends on the CallingProvider.

## Usage

```jsx
import React from 'react';
import {
  useOutgoingCall
} from '@azure/acs-ui-sdk';

const App = () => (
  <CallingProvider>
      <App />
  </CallingProvider>
);

const App = () => {
  const { makeCall, endCall, callState } = useGroupCall();
  makeCall('98885c40-4a29-11eb-a3e9-c9e18c4b8e10' /* sample acs user id */);
  ...
}
```
