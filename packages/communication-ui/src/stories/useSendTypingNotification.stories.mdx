# useSendTypingNotification

The `useSendTypingNotification` hook allows components to send a typing notification to the current thread. The hook
returns a function of the form `() => Promise<void>` which should be called when the notification should be sent. The
current thread refers to the current thread managed by ChatThreadProvider.

## Importing

```javascript
import { useSendTypingNotification } from '@azure/acs-ui-sdk';
```

## Dependencies

- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import { ChatThreadProvider, useSendTypingNotification } from '@azure/acs-ui-sdk';

const Example = () => (
  <ChatThreadProvider>
    <App />
  </ChatThreadProvider>
);

const App = () => {
  const sendTypingNotification = useSendTypingNotification();
  sendTypingNotification();
};
```
