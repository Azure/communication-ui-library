# useIsMessageSeen

The `useIsMessageSeen` hook allows components to determine if a particular message was seen. The hook returns a
function of the form `(userId: string, clientMessageId: string, messages: any[]) => boolean` which should be with the
userId, messageId, and list of messages to check. Note that this requires useSubscribeReadReceipt() to be called prior
to use.

## Importing

```javascript
import { useIsMessageSeen } from '@azure/communication-ui';
```

## Dependencies

- ChatThreadProvider

## Usage

```jsx
import React from 'react';
import { ChatThreadProvider, useIsMessageSeen } from '@azure/communication-ui';

const Example = () => (
  <ChatThreadProvider>
    <App />
  </ChatThreadProvider>
);

const App = () => {
  const isMessageSeen = useIsMessageSeen();
  const seen = isMessageSeen('userId', 'clientMessageId', []);
};
```
