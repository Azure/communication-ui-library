# useGroupCall

The `useGroupCall` hook allows components to leverage the underlying Calling JS SDK to join a ACS group call.
This hook is absolutely necessary if you want to join/leave a group call.

## Importing

```javascript
import { useGroupCall } from '@azure/communication-ui';
```

## Dependencies

- CallingProvider

## Usage

```jsx
import React from 'react';
import {
  CallingProvider,
  useGroupCall
} from '@azure/communication-ui';

const Example = () => (
  <CallingProvider token="<TOKEN>", displayName="<DISPLAYNAME>", groupId="<GROUPID>">
      <App />
  </CallingProvider>
);

const App = () => {
  const { join, leave } = useGroupCall();
  join('98885c40-4a29-11eb-a3e9-c9e18c4b8e10' /* sample group id */);
  ...
}
```
