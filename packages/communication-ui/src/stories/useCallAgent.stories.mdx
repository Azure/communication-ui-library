# useCallAgent

The `useCallAgent` hook allows us to subscribe to participant, stream, call state changes during a call. The updates are written
back to the call provider so this data can be used to the child components of the call provider. This hook is necessary for calling scenarios, as it
subscribes to call events that are important to see streams, other participants, and even the call state.

## Importing

```javascript
import { useCallAgent } from '@azure/communication-ui';
```

## Dependencies

- CallingProvider

## Usage

```jsx
import React from 'react';
import {
  CallingProvider
  useCallAgent
} from '@azure/communication-ui';

const Example = () => (
  <CallingProvider token="<TOKEN>", displayName="<DISPLAYNAME>", groupId="<GROUPID>">
      <App />
  </CallingProvider>
);

const App = () => {
  const { call } = useCallAgent();

  return <>{JSON.stringify(call)}</>
}
```
