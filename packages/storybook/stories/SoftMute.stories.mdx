# Soft Mute

<DetailedBetaBanner version={'1.19.1-beta.1'} />

The soft mute feature enables participants to mute specific or all remote participants in most call scenarios
when using CallComposite or CallWithChatComposite.

For calls that involve roles, the participant must have one of the following roles to mute remote participants: Organizer, Co-Organizer, Presenter.

## Soft Mute With Roles

### Attendee Role

Attendees do not have soft mute capabilities and will not be able to see menu options for muting another participant.

### Organizer and presenter views

Organizers, co-organizers, and presenters will see menu options for muting a single or all remote participants.
