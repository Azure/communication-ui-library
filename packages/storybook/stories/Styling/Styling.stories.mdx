# Styling

The ability to style components using custom CSS is an important feature of Azure Communication UI Toolkit.
All the UI Components have complete CSS support and provide complex styling capabilities.

## Overview

- UI Components expose a `prop` named `styles` that accepts an Object containing custom CSS for the component root as well as it's child components.

- Users can always provide the `root` property in `styles` object and apply custom CSS to the component root.

- Properties in `styles` object will vary from component to component.
